## Data Cleaning Process

- Inspected data structure
- Checked missing values
- Handled missing values
- Checked duplicate records
- Detected and evaluated outliers using the IQR method
- Standardized categorical text values
- Converted date columns to datetime
- Verified data types
- Saved the cleaned dataset

## Key Findings
- 24 missing values in the Income column were handled.
- No duplicate records were found.
- Eight income outliers were identified.
- One extreme outlier (Income = 666666) was removed after review due to inconsistency with purchasing behavior.
- Rare marital status categories were consolidated for consistency.
- The dataset is now ready for Exploratory Data Analysis (EDA).
